## 页面布局
#### 1. 假设高度已知，请写出三栏布局，其中左右栏宽度为300px，中间自适应

> 方案一：浮动float

```html
<style>
  .left {
    float: left;
    width: 300px;
  }
  .right {
    float: right;
    width: 300px;
  }
  .center {
    overflow: hidden;  /* 生成BFC */
  }
</style>

<div class="wrap clr">
  <div class="left"></div>
  <div class="right"></div>
  <div class="center"></div>
</div>
```

> 方案二：绝对定位postion: absolute

```html
<style>
  .wrap {
    postion: relative;
  }
  .wrap > div{
    postion: absolute;
    top: 0;
  }
  .left {
    left: 0;
    width: 300px;
  }
  .center {
    left: 300px;
    right: 300px;
  }
  .right {
    right: 0;
    width: 300px;
  }
</style>

<div class="wrap">
  <div class="left"></div>
  <div class="center"></div>
  <div class="right"></div>
</div>
```

> 方案三：flexbox

```html
<style>
  .wrap {
    display: flex;
  }
  .left {
    width: 300px;
    flex: none;
  }
  .center {
    flex: 1;
  }
  .right {
    width: 300px;
    flex: none;
  }
</style>

<div class="wrap">
  <div class="left"></div>
  <div class="center"></div>
  <div class="right"></div>
</div>
```

> 方案四：table-cell

```html
<style>
  .wrap {
    width: 100%;
    display: table;
  }
  .wrap > div {
    display: table-cell;
  }
  .left {
    width: 300px;
  }
  .center {
  }
  .right {
    width: 300px;
  }
</style>

<div class="wrap">
  <div class="left"></div>
  <div class="center"></div>
  <div class="right"></div>
</div>
```

> 方案五：网格布局grid

```html
<style>
  .wrap {
    display: grid;
    width: 100%;
    grid-template-rows: 100px; /* 高度 */
    grid-template-columns: 300px auto 300px;
  }
</style>

<div class="wrap">
  <div class="left"></div>
  <div class="center"></div>
  <div class="right"></div>
</div>
```
